Keysee is a minimalist input visualizer currently available for Windows only. Built for creators, streamers, and tutorial makers who want to clearly show keyboard and mouse input.

Keysee is fully open source, built with Go and Electron. You can explore the source code, contribute, or suggest features on GitHub.
